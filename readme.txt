== Description ==
This plugin is used to list all the authors based on their recent activity / Post as widget.


Note* : Please take a backup of your database before installing the plugin while installing on Production sites.

== Installation ==

1.	Unzip and Upload the authors-based-on-recent-posts folder files to the /wp-content/plugins/ directory
2.	Activate the plugin through the ‘Plugins’ menu in WordPress
3.	Navigate to Appearance –> Widgets –> Drag and Drop the “Authors Based on Recent Posts” to any widget area (Main / Second sidebar)
4.	Assign suitable title to widget based on role (Editors or Authors) and select role (authors/editors etc.)
5.	Save and close the widget
6.	Check the Authors list in blog sidebar (Main / secondary sidebar)
7.	The plugin also provide Profile style selection (Square/round )
!! The Plugin installation is completed.

 
 
== Frequently Asked Questions ==
 
Q) 	Will ABRP blog Authors conflict with my other plugins ? 
Ans: 	ABRP blog Authors works by hooking into Wordpress's widgets system and should play OK with most plugins, however there is no guarantee. 

Q) 	Does this Plugin collapses my database?
Ans: 	The plugin is no way conflicts with any other database table but it is suggested to take a backup before installing this plugin as precaution.
